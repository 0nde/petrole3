📚 How it works / Comment ça marche

1. This page calls the GitHub REST API:
POST /repos/0nde/petrole3/dispatches

2. GitHub triggers the repository-dispatch.yml workflow

3. The workflow routes to the correct job based on event_type

4. The job receives your client_payload as input data

1. Cette page appelle l'API REST GitHub
2. GitHub déclenche le workflow repository-dispatch.yml
3. Le workflow route vers le bon job selon event_type
4. Le job reçoit votre client_payload comme données d'entrée
Equivalent curl command / Commande curl équivalente:
curl -X POST \ -H "Accept: application/vnd.github.v3+json" \ -H "Authorization: token YOUR_PAT" \ https://api.github.com/repos/0nde/petrole3/dispatches \ -d '{"event_type":"ping","client_payload":{"message":"Hello!"}}'
