🚀 Événement Dispatch

📋 Journal des événements

Prêt. Entrez votre PAT et sélectionnez un type d'événement.

📚 Comment ça marche

1. Cette page appelle l'API REST GitHub :
POST /repos/0nde/petrole3/dispatches

2. GitHub déclenche le workflow repository-dispatch.yml

3. Le workflow route vers le bon job selon event_type

4. Le job reçoit votre client_payload comme données d'entrée
Commande curl équivalente :
curl -X POST \ -H "Accept: application/vnd.github.v3+json" \ -H "Authorization: token VOTRE_PAT" \ https://api.github.com/repos/0nde/petrole3/dispatches \ -d '{"event_type":"ping","client_payload":{"message":"Bonjour !"}}'
